Selected Work

SignalForge — Open-Source Signal Engine

Problem
Most open-source AI-SDR tools ship without evals, so reply rates collapse once the draft pattern-matches as AI slop. Outbound teams need signal-anchored emails tested like software, not like prompts.
Approach
Nine signal sources (Greenhouse, Ashby, Lever, GitHub, SEC EDGAR, news RSS, Exa, Hacker News, Product Hunt) feed a YAML-declared ICP scorer. Claude writes the “why now” research brief and the cold-email drafter; a reply-quality eval harness blends deterministic checks, an LLM judge, and cliff penalties so a weighted average can’t rationalize a spammy draft into a pass. Golden-set regression and a cross-model benchmark gate every prompt change.
Stack  ·  Python  ·  asyncio  ·  Streamlit  ·  Ollama  ·  Claude API  ·  SQLite
