Selected Work

Credence — Trust & Fit Scoring for Hard-to-Verify Buyers

Problem
Horizontal tools (Clay, Apollo, ZoomInfo) score B2B buyers off self-reported titles, which collapses in sensitive industries like semiconductors where actual functional scope rarely matches the LinkedIn line. Outbound and recruiting teams need an inspectable score with a full evidence trail — not another opaque firmographic.
Approach
Per-source fetchers (ATS endpoints, SEC EDGAR, USPTO, OpenAlex, GitHub, wire-service RSS, conference programs) feed Pydantic-validated extractors. A four-dimension engine — Role Fit, Authority Fit, Company Fit, Signal Confidence — returns sub-scores separately, with cited evidence IDs and required disconfirming signals to block confirmation-bias scoring. Discover is a force-directed graph across nine node types (person, company, role, city, school, conference, industry, past employer, partnership) driven by a Z.AI tool-calling chat sidebar; click a person, route into the deep-dive with the full provenance trail.
Stack  ·  React  ·  TypeScript  ·  Vite  ·  Supabase  ·  FastAPI  ·  Claude API  ·  Z.AI GLM-5.1
